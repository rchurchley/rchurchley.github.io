A brickwork fractal

2018-02-04 Math

Brick pavements and tatami mats are traditionally laid out so that no four meet at a single point to form a ┼ shape. Only a few ┼-free patterns can be made using 1×11\times 11×1 and 1×21\times 21×2 tiles, but the addition of 2×22\times 22×2 tiles provides a lot more creative flexibility.

When I discussed tatami tilings with my relative Oliver Linton, he suggested applying similar rules to other brick sizes to make beautiful tiling patterns. The tatami condition alone does not provide enough of a constraint to mathematically analyze tilings with arbitrary shapes and sizes, but it is a good starting point when looking for interesting patterns.

With the addition of 2×22\times 22×2 square tiles, it’s possible to construct rectangular blocks that fit together to tesselate the plane while preserving the four-corner rule.

This opens the door to self-similar tilings, which I’m very interested in! The goal is to use 1×11 \times 11×1, 1×21 \times 21×2, and 2×22 \times 22×2 tiles to construct n×nn\times nn×n, n×2nn \times 2nn×2n, and 2n×2n2n \times 2n2n×2n blocks which can be put together in the exact same way to make increasingly intricate nk×nkn^k \times n^knk×nk tilings that maintain the tatami condition.

The simplest non-trivial example I could find involves a set of 5×55\times 55×5, 10×1010 \times 1010×10, and two 5×105\times 105×10 rectangular tilings.

Starting with any of these four layouts, we can replace each of the 1×11\times 11×1, 2×22\times 22×2, and 1×21\times 21×2 bricks with a corresponding 5×55\times 55×5, 10×1010\times 1010×10, or 5×105\times 105×10 rectangular tiling in the correct orientation. (This will produce a few four-corner intersections, but we can fix these by merging adjacent pairs of 1×21\times 21×2 bricks.)

Repeatedly performing this operation gives an infinite sequence of tilings, but can we say they converge to anything? A tiling TTT can be identified with its outline ∂T\partial T∂T (i.e. the set of points on boundaries between two or more tiles). Note that if a point is in ∂Ti\partial T_i∂Ti​, then it will be in every subsequent ∂Tj\partial T_j∂Tj​ unless it is one of the few bricks merged in ∂Ti+1\partial T_{i+1}∂Ti+1​. So we might sensibly define the limiting object of the tiling sequence TiT_iTi​ as the union

⋃i(∂Ti∩∂Ti+1).\bigcup_i \left(\partial T_i \cap \partial T_{i+1}\right). i⋃​(∂Ti​∩∂Ti+1​).

This self-similar dense path-connected set satisfies the topological equivalent of the “four corners rule” — a pretty interesting list of mathematical properties!

The same strategy could be applied to other sets of (2n+1)×(2n+1)(2n+1)\times (2n+1)(2n+1)×(2n+1), (4n+2)×(4n+2)(4n+2)\times (4n+2)(4n+2)×(4n+2), and (2n+1)×(4n+2)(2n+1)\times (4n+2)(2n+1)×(4n+2) tiles with similar boundaries. What’s the prettiest brickwork fractal you can find?

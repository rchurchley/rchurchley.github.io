Metropolis Beamer theme

Beamer is an awesome way to make presentations with LaTeX, but its theme selection is surprisingly sparse. The stock themes share an aesthetic that can be a little cluttered, while the few distinctive custom themes available are often specialized for a particular corporate or institutional brand. Metropolis provides a simple, modern Beamer theme suitable for anyone to use. It is available on CTAN and comes bundled with the latest TeX Live distribution.

I was a major contributer to Metropolis from 2015 to 2016. If you want to help make the theme better, you can join the development efforts on Matthias Vogelgesang’s GitHub page for the project.

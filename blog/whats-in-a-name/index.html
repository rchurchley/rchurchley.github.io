What's in a name?

Ever since I first heard about it, I’ve been itching to play around with the name data released by the BC Vital Statistics Agency — a list all of the names that appear on BC’s birth certificates for each of the last hundred years! If you’re interested in playing with the data yourself, you can find them here and here.

The first thing we can dertive from the dataset is the (approximate) birth rate in BC over the last century and its notable features, including the province’s midcentury population acceleration, the baby boom, and the echo boom I was born into.

With this context in mind, we can investigate the relative frequency of a given name. (Note that data is missing for any year in which the name appears on fewer than five birth certificates.)
